# Nguyenytran Tap

## How do I install these formulae?

`brew install nguyenytran/tap/<formula>`

Or `brew tap nguyenytran/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "nguyenytran/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
